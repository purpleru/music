# 说明

- 网易云一键打卡是一个能够快速帮助您的网易云音乐账号升级的程序。
- 项目前端技术点:
  - `jQuery`
  - `Bootstrap`
  - `js-cookie`
  - `art-template`
- 项目后端技术点
  - `express`
  - `cheerio`
  - `axios`
  - `cookie`

## 开始使用

- 拉取本项目 => 下载依赖包 `npm install` => 启动项目 `npm run start` 或者 `node server.js`
- 演示地址：[http://demo.wgudu.com:3006](http://demo.wgudu.com:3006)
